## Project Overview
Developed a prototype system designed to analyze the emotional state of astronauts during space missions. Long-duration space travel can take a toll on mental health, and this system aims to provide early detection of emotional distress using AI.

## Key Features
- **Facial Emotion Recognition:** Analyzes video feeds to detect facial expressions indicating stress, happiness, or anxiety.
- **Voice Analysis:** (If applicable) Analyzes voice tone and pitch for emotional cues.
- **Real-time Monitoring:** Provides real-time feedback to mission control.
