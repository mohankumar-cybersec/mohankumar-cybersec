## Project Overview
This project involves the development of an intelligent AI agent capable of negotiating in complex scenarios. The agent leverages the **Llama-3-8B** Large Language Model (LLM) to understand context, formulate strategies, and engage in human-like negotiation dialogues.

- **LLM Integration:** Integrated the Llama-3 API to power the agent's conversational capabilities.
- **Strategy Implementation:** Implemented various negotiation strategies to optimize outcomes.
